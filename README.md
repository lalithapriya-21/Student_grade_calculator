# Student_grade_calculator
week-2 project

Project Description:-
The Student Grade Calculator is a Python-based application designed to evaluate student performance by calculating grades based on marks entered by the user. The program validates the input to ensure marks are within the range of 0–100 and assigns grades using predefined grading criteria. Along with displaying the grade, it provides encouraging messages to motivate students.
This project demonstrates the use of fundamental Python concepts such as functions, conditional statements, loops, exception handling, and string formatting. It aims to automate the grading process while ensuring user-friendly interaction and accurate results.

Features:-
Accepts student name and marks as input.
Validates marks to ensure they are between 0 and 100.
Handles invalid inputs using exception handling.
Uses a while loop to repeatedly request valid marks.
Calculates grades using if-elif-else conditions.
Displays motivational messages corresponding to each grade.
Produces clean and formatted output.

Grading System:-
Marks Range	Grade
90 – 100	A
80 – 89	B
70 – 79	C
60 – 69	D
0 – 59	F

Outcome:-
This project successfully automates grade calculation and demonstrates the practical implementation of core Python programming concepts while providing meaningful feedback to users.
